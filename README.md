# hapi-space-madness

keycloak-plugin


## Install

```bash
$ npm install hapi-space-madness-keycloak
```


## Usage

```javascript
const Plugin = 'hapi-space-madness-keycloak';
{
    plugin: {
        register: Plugin,
        options: {
            a: 1
        }
    }
},
```
test
```bash
curl http://localhost:8080/test
```
